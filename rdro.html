    • Rider's Delivery Route Optimization Using A* Algorithm

    Rider's Delivery Route Optimization Using A* Algorithm is a thesis and system application that the delivery riders can find an efficient route from nearest up to the next nearest location and the logistics company administrator can track and monitor the location of the delivery during working hours.

    Color Exploration (Riders)

    The prime color of the delivery rider's application is Light Blue. It acts for peace and travel. It is good in the eye of the riders because of this light color. It is compatible with daily work as delivery riders.

    Color Exploration (Admin)

    The prime color of the admin application is Red. It represents energy and gentleness. The respondents of the study are J&T Express. The color branding of their company is red. So I decided to copy the color red and make some customizations.

    Typography

    Open Sans is an open-source humanist sans-serif typeface. It is typically used in Android Applications. So I decided to use this font because it is popular and typically used in Android Applications.

    Logo

    There are two proposed logos which are the riders and the admin logo. The logo looks like a location icon with an eagle inside. Because eagle represents peace and freedom. The application wants the riders safety and valuable like an eagle that now is an endangered species that need to save and protect.
